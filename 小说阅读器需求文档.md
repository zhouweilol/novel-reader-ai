# 小说阅读器项目需求文档

## 一、项目概述
开发一款带有AI吐槽机器人的小说阅读器，支持网页端阅读，配合后台管理系统实现图书管理和AI配置。

---

## 二、功能需求

### 2.1 小说阅读模块
| 功能点 | 说明 |
|--------|------|
| 用户系统 | 手机号+密码注册登录（后期考虑验证码） |
| 图书展示 | 支持书架展示、分类浏览、个性化推荐 |
| 章节阅读 | 支持上下翻页、目录跳转 |
| 阅读设置 | 字体大小、背景颜色、夜间模式 |
| 阅读进度 | 自动记录上次阅读位置 |
| 搜索功能 | 按书名、作者搜索 |
| 插图功能 | 支持在章节中插入图片，每章最多5张，上传需审核 |
| 书签功能 | 支持添加/删除书签，快速跳转 |
| 自动翻页 | 支持定时自动翻页，可调节速度 |
| 章节评论 | 支持对章节发表评论 |
| 热门排行 | 基于阅读量/打赏数的排行榜 |

### 2.2 IF线续写模块
| 功能点 | 说明 |
|--------|------|
| 章末入口 | 章节阅读完毕后，AI机器人弹出对话框询问是否续写 |
| 跳过功能 | 用户可点击其他地方关闭对话框跳过 |
| AI续写 | 对未完结小说进行AI续写，约1000-2000字 |
| 读者续写 | 读者可自行创作续写内容 |
| 个人生成 | 每个用户独立生成/创作自己的续写版本 |
| 审核机制 | AI生成和读者续写均需审核通过后才能被打赏 |
| 打赏功能 | 使用账户积分打赏优秀的IF线作品 |
| 点赞功能 | 用户可以点赞喜欢的IF线作品 |
| 排行榜 | 打赏前3名公开展示，形成排行榜 |

**积分系统：**
- 新用户注册赠送初始积分
- 每日登录获取积分
- 阅读时长奖励
- 初期仅用于站内打赏，后期考虑积分交易
- 积分获取限制：防刷机制

### 2.3 吐槽机器人模块
| 功能点 | 说明 |
|--------|------|
| 侧边栏入口 | 固定位置的吐槽按钮/图标 |
| 对话框弹出 | 点击后弹出对话框界面 |
| 段落识别 | 自动识别当前阅读的段落内容 |
| AI吐槽回复 | 调用AI API生成吐槽内容 |
| 临时展示 | 吐槽内容仅临时展示，用户点击后消失，不保存历史 |

**AI接口对接：**
- 支持配置API Key
- 支持国产模型 DeepSeek / Mimo
- 吐槽prompt可配置

### 2.4 通知系统模块
| 功能点 | 说明 |
|--------|------|
| 审核通知 | IF线/插图审核结果通知 |
| 互动通知 | 被打赏、被点赞、被评论通知 |
| 系统通知 | 平台公告、活动通知 |

### 2.5 后台管理系统
| 功能点 | 说明 |
|--------|------|
| 图书管理 | 上传、编辑、删除、上下架图书 |
| 图书格式 | 支持多种格式上传：.md、.txt、.epub、.pdf、.docx，自动解析章节 |
| 章节限制 | 每章不低于2000字 |
| 用户管理 | 用户列表、权限管理、封禁 |
| AI配置 | API Key管理、模型选择（DeepSeek/Mimo）、prompt配置 |
| IF线审核 | 审核AI生成和读者续写的IF线作品（通过/拒绝） |
| 插图审核 | 审核章节插图，每章最多5张 |
| 评论管理 | 审核、删除违规评论 |
| 推荐管理 | 首页推荐位配置、热门排行管理 |
| 数据统计 | 阅读量、用户活跃度、内容统计 |

## 三、技术选型建议

### 前端
- **阅读器端**：Vue 3 / React + Vite
- **UI框架**：Element Plus / Ant Design
- **状态管理**：Pinia / Redux

### 后端
- **框架**：Node.js (Express/NestJS) 或 Python (FastAPI/Django)
- **数据库**：MySQL / PostgreSQL
- **缓存**：Redis
- **加密**：bcrypt（密码）、AES-256（敏感数据）
- **安全**：接口限流、防刷机制

### AI集成
- 国产模型：DeepSeek / Mimo
- 流式响应支持

## 四、数据库设计（初步）

**安全要求：**
- 用户密码使用 bcrypt 加密存储
- 手机号使用 AES 对称加密存储
- 小说原文档地址使用 AES 对称加密存储
- API Key使用 AES 对称加密存储

```sql
-- 图书表
CREATE TABLE books (
    id INT PRIMARY KEY,
    title VARCHAR(255),
    author VARCHAR(100),
    cover_url VARCHAR(500),
    file_path VARCHAR(500), -- 原文档地址，AES加密存储
    description TEXT,
    created_at TIMESTAMP
);

-- 章节表
CREATE TABLE chapters (
    id INT PRIMARY KEY,
    book_id INT,
    chapter_num INT,
    title VARCHAR(255),
    content TEXT,
    word_count INT, -- 每章不低于2000字
    FOREIGN KEY (book_id) REFERENCES books(id)
);

-- 用户表
CREATE TABLE users (
    id INT PRIMARY KEY,
    phone VARCHAR(255) UNIQUE, -- 手机号，AES加密存储
    password_hash VARCHAR(255), -- 密码，bcrypt加密存储
    created_at TIMESTAMP
);

-- AI配置表
CREATE TABLE ai_config (
    id INT PRIMARY KEY,
    api_key VARCHAR(500), -- API Key，AES加密存储
    model_name VARCHAR(100) DEFAULT 'deepseek', -- deepseek / mimo
    prompt_template TEXT,
    is_active BOOLEAN
);

-- IF线续写表
CREATE TABLE if_stories (
    id INT PRIMARY KEY,
    book_id INT,
    chapter_id INT,
    user_id INT,
    source ENUM('ai', 'reader') DEFAULT 'ai', -- ai生成或读者续写
    content TEXT,
    word_count INT,
    status ENUM('pending', 'approved', 'rejected') DEFAULT 'pending', -- 审核状态
    tips_count INT DEFAULT 0, -- 打赏总数
    likes_count INT DEFAULT 0, -- 点赞总数
    created_at TIMESTAMP,
    reviewed_at TIMESTAMP,
    FOREIGN KEY (book_id) REFERENCES books(id),
    FOREIGN KEY (chapter_id) REFERENCES chapters(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 用户积分表
CREATE TABLE user_points (
    id INT PRIMARY KEY,
    user_id INT UNIQUE,
    points INT DEFAULT 0,
    updated_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 打赏记录表
CREATE TABLE tips_records (
    id INT PRIMARY KEY,
    user_id INT,
    if_story_id INT,
    points INT,
    created_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (if_story_id) REFERENCES if_stories(id)
);

-- 点赞记录表
CREATE TABLE likes_records (
    id INT PRIMARY KEY,
    user_id INT,
    if_story_id INT,
    created_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (if_story_id) REFERENCES if_stories(id),
    UNIQUE(user_id, if_story_id) -- 防止重复点赞
);

-- 章节插图表
CREATE TABLE chapter_images (
    id INT PRIMARY KEY,
    chapter_id INT,
    user_id INT,
    image_url VARCHAR(500), -- 图片地址，AES加密存储
    position INT, -- 图片在章节中的位置（段落序号）
    status ENUM('pending', 'approved', 'rejected') DEFAULT 'pending', -- 审核状态
    created_at TIMESTAMP,
    reviewed_at TIMESTAMP,
    FOREIGN KEY (chapter_id) REFERENCES chapters(id),
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 书签表
CREATE TABLE bookmarks (
    id INT PRIMARY KEY,
    user_id INT,
    chapter_id INT,
    position INT, -- 书签位置（段落序号）
    created_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (chapter_id) REFERENCES chapters(id)
);

-- 章节评论表
CREATE TABLE comments (
    id INT PRIMARY KEY,
    user_id INT,
    chapter_id INT,
    content TEXT,
    status ENUM('pending', 'approved', 'rejected') DEFAULT 'approved',
    created_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (chapter_id) REFERENCES chapters(id)
);

-- 通知表
CREATE TABLE notifications (
    id INT PRIMARY KEY,
    user_id INT,
    type ENUM('review', 'interaction', 'system'), -- 审核/互动/系统
    title VARCHAR(255),
    content TEXT,
    is_read BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- 阅读记录表
CREATE TABLE read_history (
    id INT PRIMARY KEY,
    user_id INT,
    book_id INT,
    chapter_id INT,
    read_time INT DEFAULT 0, -- 阅读时长（秒）
    created_at TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (book_id) REFERENCES books(id),
    FOREIGN KEY (chapter_id) REFERENCES chapters(id)
);
```

## 五、API设计（初步）

### 阅读器端API
- `POST /api/auth/register` - 用户注册（手机号+密码）
- `POST /api/auth/login` - 用户登录
- `GET /api/books` - 获取图书列表
- `GET /api/books/:id` - 获取图书详情
- `GET /api/books/:id/chapters` - 获取章节列表
- `GET /api/chapters/:id` - 获取章节内容
- `POST /api/if-stories/generate` - 生成IF线续写（AI）
- `POST /api/if-stories/submit` - 提交IF线续写（读者）
- `GET /api/if-stories/:chapterId` - 获取章节的IF线列表（仅已审核）
- `POST /api/if-stories/:id/tip` - 打赏IF线作品
- `POST /api/if-stories/:id/like` - 点赞IF线作品
- `GET /api/if-stories/ranking` - 获取打赏排行榜
- `POST /api/chapters/:id/images` - 上传章节插图
- `GET /api/chapters/:id/images` - 获取章节插图列表（仅已审核）
- `POST /api/bookmarks` - 添加书签
- `DELETE /api/bookmarks/:id` - 删除书签
- `GET /api/bookmarks` - 获取用户书签列表
- `POST /api/chapters/:id/comments` - 发表评论
- `GET /api/chapters/:id/comments` - 获取章节评论列表
- `GET /api/notifications` - 获取通知列表
- `PUT /api/notifications/:id/read` - 标记通知已读
- `GET /api/read-history` - 获取阅读历史
- `POST /api/read-history` - 更新阅读时长
### 后台管理API
- `POST /api/admin/books/upload` - 上传图书（支持.md、.txt、.epub、.pdf、.docx格式）
- `PUT /api/admin/books/:id` - 编辑图书
- `DELETE /api/admin/books/:id` - 删除图书
- `POST /api/admin/ai-config` - 配置AI参数
- `GET /api/admin/if-stories` - 获取待审核IF线列表
- `PUT /api/admin/if-stories/:id/review` - 审核IF线作品（通过/拒绝）
- `GET /api/admin/images` - 获取待审核插图列表
- `PUT /api/admin/images/:id/review` - 审核插图（通过/拒绝）
- `GET /api/admin/comments` - 获取待审核评论列表
- `PUT /api/admin/comments/:id/review` - 审核评论（通过/删除）
- `POST /api/admin/recommend` - 配置首页推荐
- `GET /api/admin/stats` - 获取统计数据
